# BPERS Bludit Themes

Documentation

How to work and install Bpers News Themes template on your hosting

[Download BPers Project →](https://bludit.axcora.com/bpers/download)

Login in to your hosting panel - Upload and Extract Bpers Source Code files into your Website Project

Access Your site and Run Installation

Insert your password , by default bludit is use admin as user, but you can change it later for security reason

Finish Your Bpers News Themes is live on your website.

For login in to admin panel area , you can visit on yoursite/admin , then input your password and click login.

For security reason you can access user menu, add new user set as admin, insert user ID and save it, next delete your default admin user.

Setup your site - access the settings - general - enter the title, description, slogan and so on, adjust it to your needs, then you can click on another tab and configure it - For the image header area you can upload your image header on the logo tab settings.

Activate Plugins - access on Settings - Plugins - and you can activate this plugins list for help improve your SEO.

+ RSS Feed
+ Sitemap
+ Robots
+ Canoncial
+ Open Graph
+ Twitter Card
+ Search

and others what you needs just activate the plugins.

Create categories for grouping your article content - add new categories and insert title name and description - save.

Create Blog Post - Click on New Content and insert title, write your content, next you can click on options button on content area menu - select categories - add your description , upload cover image. for add tags you can click on options button - advanced tabs - insert your tags, select type default for blog post or dynamic page type.

Edit and Delete Post and Static Page- Click on Content, and you can click edit or delete your post.

Create a Static Page - Click New Content - Insert a title, and write your article, next you can click the options button in the header content area, select a category - enter your description, then swipe on the advanced tab - on the form type set as static, then enter the tag , and save.

If you want make your site more attractive and complete, you can use BPERS widget,

[Check Custom Plugins Widget Now →](https://bludit.axcora.com/bpers/widget)
